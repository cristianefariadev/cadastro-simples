# Cadastro Básico em JSP/SERVLET

Formulário de cadastro de:

* clientes
* fornecedores
* produtos


## Tecnologias utitilizadas

* JAVA EE
* TOMCAT
* JSP/SERVLET
* Persistência JDBC
* POSTGRESQL
* HTML5
* CSS3
* BOOSTRAP
* JAVASCRIPT
* JQUERY
